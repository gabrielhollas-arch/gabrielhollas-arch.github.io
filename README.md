
# Meu Catálogo

## Descrição
Este é um sistema de **Catálogo de Produtos**, onde os usuários podem visualizar uma lista de produtos cadastrados. Cada produto contém informações como nome, preço, categoria e imagem. Os dados são armazenados no **Supabase**, uma plataforma de banco de dados como serviço, e carregados dinamicamente na página.

O catálogo foi criado como parte de um projeto de demonstração para aprender a integrar **HTML**, **CSS**, **JavaScript** e **Supabase**.

## Funcionalidades
- Visualização de produtos com nome, preço, categoria e imagem.
- Produtos são carregados dinamicamente de um banco de dados Supabase.
- Preço é formatado corretamente no formato da moeda brasileira (BRL).
- Interface simples e funcional para exibição dos dados.

## Tecnologias Utilizadas
- **HTML**: Para estruturar o conteúdo da página.
- **CSS**: Para estilizar os componentes e garantir uma boa aparência do catálogo.
- **JavaScript**: Para lógica de interação, como carregar os dados dinamicamente do Supabase.
- **Supabase**: Plataforma de banco de dados que armazena e fornece os dados do catálogo.
- **GitHub Pages**: Para hospedar o projeto online.

## Como Executar
1. Clone este repositório para o seu computador.
2. Abra o arquivo `index.html` em um navegador para visualizar o catálogo.
3. Certifique-se de ter uma conta no Supabase e configurar seu banco de dados para que o catálogo funcione corretamente.
